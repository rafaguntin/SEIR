# SEIR model

SEIR models the flows of people between four states: susceptible (S), exposed (E), infected (I), and resistant (R).

## States and parameters

SEIR has 7 states:
1) Suceptible $$S_t$$
2) Insusceptibles
3) Exposed
4) Infectious
5) Quarantined
6) Recovered
7) Dead
